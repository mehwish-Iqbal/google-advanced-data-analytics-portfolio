# 🦄 Unicorn Companies Analysis (Portfolio Project)

## 📌 Overview

This project analyzes a dataset of **1,074 unicorn companies** to understand growth patterns, valuation trends, and investment behavior. The goal is to extract meaningful business insights using Python-based data analysis and visualization techniques.

---

## 🎯 Objectives

* Analyze time taken by companies to achieve unicorn status
* Study valuation differences across industries
* Identify high-value companies and investment patterns
* Generate data-driven business insights for decision-making

---

## 🛠️ Tools & Technologies

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 🔄 Project Workflow

### 1️⃣ Data Import

* Loaded dataset using Pandas
* Imported required libraries for analysis and visualization

### 2️⃣ Data Exploration

* Checked dataset structure and size
* Identified missing values and data types
* Explored key columns like Industry, Valuation, Country, Date Joined

### 3️⃣ Data Cleaning & Feature Engineering

* Converted date columns into proper datetime format
* Extracted **Year Joined** from Date Joined column
* Created new feature:

  * `Years to Unicorn = Year Joined - Year Founded`

### 4️⃣ Exploratory Data Analysis (EDA)

* Analyzed distribution of unicorn companies by industry
* Compared time-to-unicorn across sectors
* Studied valuation trends by industry

### 5️⃣ Data Visualization

* 📊 Bar chart: Time taken to reach unicorn status by industry
* 📊 Bar chart: Maximum valuation per industry

## 📊 Key Insights

* Not all unicorn companies achieve success quickly
* Some companies take longer but reach extremely high valuations
* Industry plays a major role in valuation and growth speed
* Long-term growth can be equally powerful as rapid scaling

---

## 💡 Final Business Insight

The dataset contains **1,074 unicorn companies**. Analysis shows that some companies took longer to achieve unicorn status but still reached very high valuations by March 2022. This indicates that rapid growth is not the only path to success. Companies with longer development cycles can also generate significant value.

For investment strategy, focusing on **industry trends and companies with strong valuation but fewer investors** may reveal high-potential opportunities.
