# Foundations of Data Science

## Overview

This repository contains my coursework, exercises, and learning activities completed as part of the Foundations of Data Science course from the Google Advanced Data Analytics Professional Certificate.

The course introduced fundamental concepts of data science, including data exploration, data visualization, descriptive statistics, ethical data practices, and the PACE workflow. Through hands-on activities and real-world datasets, I developed practical skills in analyzing and interpreting data to support data-driven decision-making.

---

## Course Objectives

1. Understand the foundations of data science and analytics.
2. Learn the data science workflow and its applications.
3. Explore datasets using exploratory data analysis (EDA).
4. Apply descriptive statistics to summarize data.
5. Create visualizations to communicate insights.
6. Understand data ethics and responsible data use.
7. Develop problem-solving skills using data.

## PACE Workflow

### Plan

* Define business problems and objectives.
* Identify data requirements.
* Establish project goals.

### Analyze

* Explore datasets.
* Clean and prepare data.
* Identify patterns and trends.
* Perform descriptive statistical analysis.

### Construct

* Create visualizations.
* Develop analytical solutions.
* Generate meaningful insights.

### Execute

* Communicate findings effectively.
* Present recommendations.
* Support data-driven decision-making.

By completing this course, I gained foundational knowledge in data science and developed practical skills for exploring, analyzing, and visualizing data. I also learned how to apply the PACE framework to structure data projects and communicate insights effectively.
